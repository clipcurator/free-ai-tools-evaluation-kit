# Free AI Tools Evaluation Kit

English | [简体中文](README.zh-CN.md)

Evaluation checklists for free AI tools, freemium limits, usage caps, watermarks, commercial rights, and upgrade triggers.

This repository is a neutral, practical resource for free AI tools evaluation, freemium plan limits, usage caps, output rights, and upgrade decision criteria. It is designed to help teams make clearer decisions, write better review notes, and avoid vague tool or workflow choices.

## Why This Repository Exists

Free AI tools are useful for testing, but free plans often have limits around usage, watermarks, quality, commercial rights, and exports. A checklist helps users test before committing.

## Who This Is For

- users comparing free AI tools
- directory writers explaining freemium limits
- teams testing AI tools before purchase

## Core Workflow

1. Define the decision or review goal
2. Collect the required product, content, or workflow inputs
3. Use one template to make the evaluation comparable
4. Write fit, limits, and next-action notes
5. Refresh the document when products or workflows change
